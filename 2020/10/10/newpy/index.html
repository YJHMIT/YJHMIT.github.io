毕竟我蒟蒻。。构建，调试和优化都花了比较长的时间。当然建议大家自行解决了。

#10-10

最开始的时候看了很多别人的源码啊，就是之前的大佬们整出来的东西，但是好长好长，总是没耐心看完，毕竟说实话，代码风格多样，行数又多，函数板块也不少，就很零散（还没注释）。不得不说，读程真的绝了….

后来摸索摸索就开始自己写，还有前人的思路前期点拨一把，行动起来快了很多。最开始明确了一下函数板块吧，预处理命题公式，得到输出真值表的行列数，我后期调用了堆栈去解决美观输出真值表的问题。紧接着请把命题依次用TF替换掉，便于后期后缀计算。然后就是数据结构里讲过的，中缀转后缀，不多说了，在后缀计算里，应该注意把TF又转换为10来计算最后的真值，再次调用10转TF得到命题公式的真值表。最后根据真值表就整两个循环把主析取范式和主合取范式搞出来了。

现在还不是很会markdown画图。。。

啧半个多小时过去了，为了给出一个更好的呈现呢，又把markdown绘制框图学了一下。编辑器里还差点插件，查缺补漏的又整了一下。凑合看吧有点丑，其实只是想说明顺序罢了。

主函数中输出真值表的时候，为了输出正常格式的TF，确实花了不少时间去优化，本身的代码没问题，主要是为了好看（bushi…

最开始走了不少弯路，代码分块实现之后，就是无法衔接在一起。是大佬点拨一番才领悟到最开始的方向有问题的。好在分块实现其实跟大佬的逻辑相差不多，还在此基础上优化了一番，算作满意。

总的说来，转换思想非常非常重要，循环的使用和栈的使用也必不可少。只放主函数，后期等离散结束了代码同步更新至github，感谢各路大佬的倾情奉献~~

python int main() { cout << "请使用 ! | & => 代替 非 析取 合取 单条件" << endl; string input;//输入字符 cin >> input; vector<string> varName;//向量组，也类似于容器，存放字符 parseInput(input, varName); int varNum = varName.size(); std::stack<int> s1; int tableNum = (int)pow(2, varNum);//建立真值表行数为2的n次方，n为输入命题的个数 //这里的varNum由预处理计算出来 bool table[tableNum][varNum + 1]; //多一行用来表示当前的表达式的真值 cout << "真值表" << endl; //差点给我整吐了，调到最后发现栈真的是绝了，一击致命 for (int i = 0; i < varNum; i++) { cout << varName[i] << '\t'; } for (int i = 0; i < tableNum; i++) { int num = i; for (int j = 0; j < varNum; j++) { //table[i][j] = num % 2; s1.push(num % 2); num /= 2; } } for(int i=0;i<tableNum;i++){ for(int j=0;j<varNum;j++){ table[i][j]=s1.top(); s1.pop(); } } //把预处理之后得到的命题字符向量组输出 cout << input << endl;//输出当前的命题操作 for (int i = 0; i < tableNum; i++) { string expr = input; for (int j = 0; j < varNum; j++) { stringReplace(expr, varName[j], boolToString(table[i][j])); //把expr中原命题所在的位置用TF替换掉，便于后期计算 cout << boolToString(table[i][j]) << "\t"; //挨个输出真值，得到前半段单命题的真值表 } string suffixExpr = inffixToSuffix(expr);//得到后缀表达式 table[i][varNum] = calcSuffixExpr(suffixExpr); //计算出表达式的值赋值给当前行的最后一个位置，也就是表达式的真值 if(table[i][varNum]) cout << "T" << endl;//输出 else cout<<"F"<<endl; } cout << "主合取范式" << endl; bool first =true; for (int i = 0; i < tableNum; i++) {//按行下移，找真值 if (table[i][varNum]) { continue; } if (first) { first = false; } else { cout << "&";//此处输出小项之间的合取符 } cout << "("; for (int j = 0; j < varNum; j++) { if (table[i][j]) { cout << "!"; } cout << varName[j];//输出此列的命题 if (j != varNum - 1) { cout << "|";//组成小项的关系应该是析取 } } cout << ")"; } cout << endl; cout << "主析取范式" << endl; first = true; for (int i = 0; i < tableNum; i++) { if (table[i][varNum] == false) { continue; } if (first) { first = false; } else { cout << "|"; } cout << "("; for (int j = 0; j < varNum; j++) { if (table[i][j] == false) { cout << "!"; } cout << varName[j]; if (j != varNum - 1) { cout << "&"; } } cout << ")"; } cout << endl; return 0; }
